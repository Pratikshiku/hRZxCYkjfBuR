### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页轮播图
4. 个人中心（修改个人信息、头像、修改密码）
5. 电动车检索
6. 电动车租赁
7. 租赁公告查看
8. 电动车信息一览
9. 电动车租赁历史
10. 订单信息历史
11. 电动车报修

**管理员功能：**
1. 登录
2. 个人中心（修改个人信息、修改密码）
3. 用户管理
4. 电动车信息管理
5. 电动车租赁管理
6. 订单信息管理
7. 电动车报修管理
8. 系统管理
